The Statistics Package is the core that allows loadUI to show statistics in the StatisticsWindow. The package revolves around the following classes:

    {@link StatisticVariable}

    Statistic

    Datapoint

    A datapoint represents a single 2D data point, having a timestamp and a Number value.

    StatisticsWriter

    Writes statistics data to a Track (which eventually ends up in the H2 database). A StatisticsWriter can expose several Statistics.

    StatisticHolder

    ' A Holder of Statistics (typically a loadUI Component or Testcase ). Fires CollectionEvents for contained StatisticVariables.

    StatisticManager

    A registry of available Statistics and StatisticHolders, and the settings they use. Fires a CollectionEvent when StatisticHolders are added or removed.
